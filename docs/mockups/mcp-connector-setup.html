MCP integration setup

Connect Nexus to external AI agents (Claude Desktop, Codex, Cursor, and more) so they can read and write your vault.

🔌

External agents talk to your vault through a small bridge file, connector.js, that Nexus generates inside this plugin's folder. The one-click buttons below create it and wire it into each agent's config for you — you don't need to make it yourself. Prefer to do it by hand? Every agent has a “Set up manually” section with numbered steps.
